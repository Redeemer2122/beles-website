# BELES Engineering Playbook v1.0 — Studio Edition

This package contains 4 concise but detailed Markdown documents:

1. `01-Strategy-Experience.md` — positioning, IA, homepage experience script, page blueprints, content strategy.
2. `02-Design-Motion-Media.md` — design DNA, reference translation, motion rules, GSAP guidance, media strategy.
3. `03-Technical-Implementation.md` — Astro architecture, i18n, styling, SEO, accessibility, performance, CMS future, Codex workflow.
4. `04-Hybrid-Art-Asset-System.md` — managed workflow for complex art-directed visuals without hardcoding or fragile CSS decoration.

These documents are intentionally practical and not inflated for page count.
